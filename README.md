# student-portal

UNICROSS CSC 1/4 Student Portal
NASCO is a web-based student class portal built for the Computer Science department (1/4) at the University of Cross River State (UNICROSS), serving the 2025/2026 academic session.
Designed to replace scattered WhatsApp messages and notice board updates, NASCO gives all 398 students in the class one clean, fast place to access everything academic.
Features
📚 Full course schedule with day-based filtering
📖 Detailed course pages — topics, lecturer info, and assignment tracker
📢 Announcements with mark-as-read and dismiss
👥 Class rep profiles with contact details
📅 Weekly timetable and upcoming events
📊 Overview stats — students, courses, year, semester
🔔 Toast notifications for all interactions
Built With
React (via CDN — no build step)
Plain CSS with CSS custom properties
Google Fonts — Playfair Display, DM Sans, DM Mono
Hosted on GitHub Pages
